# nonapx

> 口腔颌面外科的低冗余影像生成方案

## Introduction

#### 螺旋 CT 检查的必要性

在大型综合医院及口腔颌面外科中心，螺旋 CT 是颌骨占位性病变患者的常规影像检查。相较于锥形束 CT（CBCT），螺旋 CT 具有更好的软组织分辨率，并可结合增强扫描评估肿瘤浸润范围、血管关系及周围软组织受累情况。CBCT 主要面向高密度骨性结构显示，虽然在牙槽骨、牙根、阻生牙和颌骨骨质评估中具有优势，但软组织分辨率有限，难以准确判断占位性病变边界。因此，在颌骨肿瘤、囊肿及复杂颌面外科病例中，螺旋 CT 往往比 CBCT 更具综合诊断价值。

#### 额外 PX 检查的问题

螺旋 CT 通常不具备牙科全景片（panoramic X-ray, PX）的专用采集功能。患者即使已经完成高分辨率颌面部 CT 检查，仍可能需要额外接受一次 PX 检查，以获得临床医生熟悉的牙弓展开视图、牙列整体关系及颌骨二维全景观察信息。与部分口腔专科设备可在 CBCT/PX 一体机中完成联合采集不同，综合医院场景下的 CT 与 PX 往往来自不同设备、不同时间和不同体位。这不仅增加检查流程和辐射暴露，也导致 CT 与 PX 之间难以建立严格的空间对应关系。

因此，若能基于既有螺旋 CT 自动生成符合临床阅片习惯的 PX 图像，将有助于减少重复检查和额外辐射暴露，并在不增加患者检查负担的前提下提供牙弓整体评估所需的全景影像信息。

#### 结构准确性与模态差异

从螺旋 CT 生成 PX 并非单纯的图像风格转换，而同时包含三维结构重组、二维投影生成和跨模态观测转换。螺旋 CT 提供了较高空间定位精度和完整三维解剖信息，但 PX 是沿牙弓轨迹展开后的二维投影视图，其成像结果受牙弓曲线、全景扫描轨迹、焦层厚度、头颈姿态、上下颌位置、咬合状态及设备后处理共同影响。因此，即使同一患者同时具有 CT 与 PX，两者之间也难以建立严格的像素级对应关系。

在这一问题中，结构准确性应主要由 CT 体数据及可解释的几何投影过程保证，而真实 PX 的灰度分布、模糊、噪声、背景投影和设备后处理则属于观测域差异。由此，CT 到 PX 的生成过程更适合分解为结构投影与模态转换两个层次，而不宜直接建模为端到端的二维图像翻译问题。

#### CycleGAN 模型的结构保真局限

无配对图像模态翻译的经典方法包括 CycleGAN 及其变体。此类方法通过双向映射和循环一致性约束，在不依赖严格成对样本的条件下实现源域到目标域的转换。然而，循环一致性并不等价于解剖结构保真。模型可能在 SimPX 到 PX 的转换过程中改变局部牙齿、牙根、骨皮质或病灶边界，再通过反向网络恢复原始结构，从而满足循环损失，却无法保证中间生成图像在临床结构层面可靠。

对于口腔颌面影像而言，这类隐式结构漂移可能掩盖牙列、骨缺损、占位边界、阻生牙或金属修复体等关键结构差异，限制其作为临床影像替代或辅助工具的可信度。因此，CT 到 PX 的生成模型需要在跨模态转换的同时显式约束结构一致性，并尽可能保留从 CT 到模拟投影的可解释成像链路。

#### 基于分布桥接模型的 CT-PX 图像生成

基于上述考虑，从螺旋 CT 全自动生成全景片 PX 可分解为三个连续过程：

1. 3D-3D：从 CT 生成沿牙弓曲线展开的全景体积 PanoCT
2. 3D-2D：从 PanoCT 投影为模拟全景片 SimPX
3. 2D-2D：从 SimPX 生成符合真实全景片风格的 PX

PanoCT 是由原始 CT 重采样得到的牙弓展开 3D 体数据。其坐标系 X 轴对应牙弓曲线的切线方向，即 PX 的横向展开方向；Y 轴指向颌面部上方，即 PX 的纵向上方；Z 轴对应 X-ray 透视方向，即 PX 的深度累计方向。由此，PanoCT 保留了 CT 的三维结构信息，同时将其重构到更接近全景片成像过程的轨迹对齐空间。

流桥（Flow/Bridge Matching）模型能够在源域与目标域之间学习确定性较强的低熵传输路径，并将图像特征从源分布推向目标分布。因此，采用 3D 流桥模型实现第一阶段的 CT-PanoCT 转换，以及 2D 流桥模型实现第三阶段的 SimPX-PX 转换。第一阶段属于 3D-3D 结构转换，其目标是隐式学习经验最优的牙弓展开方式和全景扫描轨迹；第三阶段属于 2D-2D 模态转换，其目标是在保持结构一致性的前提下生成符合真实 PX 观测特征的图像。

第二阶段采用固定的透视叠加算法，PanoCT 沿 Z 轴经过带权累计投影生成 SimPX。此过程不引入神经网络模型，而是作为确定性的物理投影算子，用于维持 CT 到 SimPX 的结构可解释性。

在人工标注阶段，通过交互式微调 CT 中的牙弓曲线和全景扫描轨迹，重采样得到 PanoCT，并进一步投影生成 SimPX，直到 SimPX 与真实 PX 图像中的牙列、骨皮质、骨缺损、占位、金属修复体等关键结构特征达到近似对齐。由此构建配对的 CT-PanoCT 数据集和 SimPX-PX 数据集。第一阶段训练中，3D 流桥模型从 CT-PanoCT 数据集中隐式学习经验最优的牙弓曲线和全景扫描轨迹，避免显式回归轨迹参数带来的建模约束。第三阶段训练中，2D 流桥模型学习多尺度、结构保持的图像映射规律，使最终生成的 PX 在灰度分布、模糊、噪声、背景投影、后处理特征等方面更接近真实全景片模态，从而符合医生的阅片习惯。
