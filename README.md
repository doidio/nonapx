# Nonapx

> 口腔颌面外科 CT 与全景片 PX 图像生成

## Introduction

#### 螺旋 CT 检查的必要性

在大型综合医院及口腔颌面外科中心，螺旋 CT 是颌骨占位性病变患者的常规影像检查。相较于锥形束 CT（CBCT），螺旋 CT 具有更好的软组织分辨率，并可结合增强扫描评估肿瘤浸润范围、血管关系及周围软组织受累情况。CBCT 主要面向高密度骨性结构显示，虽然在牙槽骨、牙根、阻生牙和颌骨骨质评估中具有优势，但软组织分辨率有限，难以准确判断占位性病变边界。因此，在颌骨肿瘤、囊肿及复杂颌面外科病例中，螺旋 CT 往往比 CBCT 更具综合诊断价值。

#### 额外 PX 检查的问题

螺旋 CT 通常不具备牙科全景片（panoramic X-ray, PX）的专用采集功能。患者即使已经完成高分辨率颌面部 CT 检查，仍可能需要额外接受一次 PX 检查，以获得临床医生熟悉的牙弓展开视图、牙列整体关系及颌骨二维全景观察信息。与部分口腔专科设备可在 CBCT/PX 一体机中完成联合采集不同，综合医院场景下的 CT 与 PX 往往来自不同设备、不同时间和不同体位。这不仅增加检查流程和辐射暴露，也导致 CT 与 PX 之间难以建立严格的空间对应关系。

因此，若能基于既有螺旋 CT 自动生成符合临床阅片习惯的 PX 图像，将有助于减少重复检查和额外辐射暴露，并在不增加患者检查负担的前提下提供牙弓整体评估所需的全景影像信息。

#### 结构准确性与模态差异

从螺旋 CT 生成 PX 并非单纯的图像风格转换，而同时包含三维结构重组、二维投影生成和跨模态观测转换。螺旋 CT 提供了较高空间定位精度和完整三维解剖信息，但 PX 是沿牙弓轨迹展开后的二维投影视图，其成像结果受射线扫描轨迹、焦层厚度、头颈姿态、上下颌位置、咬合状态及设备后处理共同影响。因此，即使同一患者同时具有 CT 与 PX，两者之间也难以建立严格的像素级对应关系。

在这一问题中，结构准确性应主要由 CT 体数据及可解释的几何投影过程保证，而真实 PX 的灰度分布、模糊、噪声、背景投影和设备后处理则属于观测域差异。由此，CT 到 PX 的生成过程更适合分解为结构投影与模态转换两个层次，而不宜直接建模为端到端的二维图像翻译问题。

#### CycleGAN 模型的结构保真局限

无配对图像模态翻译的经典方法包括 CycleGAN 及其变体。此类方法通过双向映射和循环一致性约束，在不依赖严格成对样本的条件下实现源域到目标域的转换。然而，循环一致性并不等价于解剖结构保真。模型可能在 SimPX 到 PX 的转换过程中改变局部牙齿、牙根、骨皮质或病灶边界，再通过反向网络恢复原始结构，从而满足循环损失，却无法保证中间生成图像在临床结构层面可靠。

对于口腔颌面影像而言，这类隐式结构漂移可能掩盖牙列、骨缺损、占位边界、阻生牙或金属修复体等关键结构差异，限制其作为临床影像替代或辅助工具的可信度。因此，CT 到 PX 的生成模型需要在跨模态转换的同时显式约束结构一致性，并尽可能保留从 CT 到模拟投影的可解释成像链路。

## Methods

#### CT-SimPX 全景扫描轨迹与射线几何生成

全景片 PX 可视为 CT 空间中射线源沿扫描轨迹连续运动，并按扫描相位逐列采样、拼接得到的二维投影图像。PX 横轴对应扫描相位及其局部切线方向，PX 纵轴对应患者颌面的上下方向。在任一扫描相位 $t$，射线源经准直窄缝形成纵向扇束，穿过 CT 体积后在探测器上形成一条窄带投影。随 $t$ 连续变化，窄带投影依次排列，构成完整全景图像。

以相位 $t$ 的射线源位置为原点，在 CT 空间中建立局部直角坐标系 $(X_t,Y_t,Z_t)$ 。其中， $X_t$ 为扫描轨迹切线方向，对应 PX 横向相位变化。 $Y_t$ 为竖直向上方向，对应 PX 纵轴。 $Z_t$ 为扇束中心射线方向，对应射线传播及深度叠加方向。在此局部坐标系下，CT 中任意体素可表示为坐标 $(x_t,y_t,z_t)$ 。其中， $x_t$ 表示体素到当前扇束中心平面的有向距离， $y_t$ 表示体素在扇束平面内的竖直位置， $z_t$ 表示体素沿射线方向的投影深度。

因此，单一相位 $t$ 的射线几何可表示为定义在原始 CT 空间中的多通道隐式场，完整扫描轨迹则由连续相位上的隐式场共同描述。可训练 3D 条件流匹配模型，以 CT 图像作为空间条件输入，以归一化扫描相位 $t$ 作为全局条件并通过交叉注意力注入，生成该相位对应的射线几何隐式场。随后，利用距离加权拟合与局部坐标反演，可从隐式场中恢复扫描轨迹、扇束中心平面及各相位的局部射线坐标系。

需要指出的是，该模型学习的并非设备内部不可观测的真实机械轨迹，而是在 CT 空间中、以真实 PX 结构对齐结果为参照得到的经验扫描轨迹。具体而言，收集同一患者的 CT 和 PX 图像，在人工标注阶段交互式微调 CT 空间中的全景扫描轨迹，反复合成 SimPX，并对照真实 PX 中的牙列、骨皮质、骨缺损、占位及金属修复体等关键结构进行校正，直至达到可接受的一致性。由此得到的轨迹和射线几何隐式场可作为 3D 条件流匹配模型的弱监督几何目标。

为提高模型对轨迹不确定性及局部分布形态的刻画能力，可在经验轨迹附近加入平滑、受约束的几何扰动，生成不同偏离程度的训练样本。扰动应保持扫描轨迹连续性、相位顺序、扇束方向稳定性及源探测器几何一致性。该扩充过程不仅增加了训练样本数量，也为流匹配模型提供了经验轨迹邻域内的几何分布样本，使其能够学习真实可接受轨迹与偏离轨迹之间的连续变化规律。将扰动强度 $\lambda$ 作为全局条件注入模型，可进一步建立轨迹分布与偏离程度之间的条件关系。推理阶段取 $\lambda=0$，以生成尽可能接近经验标注的全景扫描轨迹。

#### SimPX-PX 模态转换

基于 CT 图像及其全景扫描轨迹，采用固定的透视叠加算法，即可合成模拟全景片 SimPX，作为后续真实全景片模态转换的源图像。

流匹配模型不仅可以学习从噪声域到图像域的生成路径，也可在两个图像域之间学习确定性较强的低熵传输路径，将源域分布逐步推向目标域分布。人工标注阶段构建的弱配对 CT-SimPX-PX 数据集，可为流匹配模型提供跨域对应关系约束。因此，可采用 2D 流匹配模型学习多尺度、结构保持的 SimPX-PX 图像映射，在保持牙列、颌骨轮廓、病灶及修复体等结构一致性的前提下，校正灰度分布、成像模糊、噪声、背景投影及设备后处理特征，使生成图像更接近真实全景片模态，并符合临床阅片习惯。

## Codex 评价以上研究思路

#### 相关论文

最接近本方案的是 2024 年的 *Panorama Tomosynthesis from Head CBCT with Simulated Projection Geometry*。该文指出，传统方法依赖牙弓曲线展开，而牙弓提取缺乏统一金标准；因此转向模拟投影几何、动态旋转中心、焦层和 Beer-Lambert 线积分生成全景图像，并在 600 例 CBCT 上验证，覆盖缺牙和金属植入场景。不过，该工作的扫描轨迹仍主要依赖显式规则设计，而不是从真实 PX 对齐中学习经验轨迹。本方案中的 3D 条件流匹配生成经验扫描轨迹，可视为在这一方向上的进一步扩展。论文链接：[arXiv 2408.09358](https://arxiv.org/abs/2408.09358)。

NeBLa 将 SimPX 作为中间表示，用 CBCT 经过 Beer-Lambert 定律和全景旋转成像原则生成 SimPX，再将真实 PX 转换到 SimPX 风格后进行 3D 重建。这支持了将 SimPX 作为结构中间域的思路，但 NeBLa 的主要方向是 PX 到 3D 的逆重建，而不是从 CT 生成真实 PX 样式图像。论文链接：[arXiv 2304.04027](https://arxiv.org/abs/2304.04027)。

Oral-3Dv2 强调仅依赖投影图和 X 射线管轨迹来学习隐式 3D 表示，说明扫描轨迹在全景影像任务中是核心变量。不过，该工作仍属于 PX 到 3D 的逆问题。论文链接：[arXiv 2303.12123](https://arxiv.org/abs/2303.12123)。

医学物理方向的 *Generating spectral dental panoramic images from single energy CT volumes* 证明了从 CT 体数据生成合成全景图在物理仿真上具有可行性，并使用头模和实验全景图进行交叉验证。但该工作尚不是患者级临床生成框架，也没有引入基于真实 PX 对齐的经验轨迹学习。论文链接：[arXiv 2305.10087](https://arxiv.org/abs/2305.10087)。

流匹配方法本身具有方法学依据。*Flow Matching for Generative Modeling* 通过回归向量场学习概率路径，并支持非扩散路径和最优传输路径。Latent Bridge Matching 进一步将桥匹配思想用于图像到图像转换。这些工作支持将 SimPX-PX 建模为 2D 流匹配或流桥问题。论文链接：[Flow Matching](https://arxiv.org/abs/2210.02747)、[Latent Bridge Matching](https://arxiv.org/abs/2503.07535)。

#### 创新性评价

本方案的创新性较强，但创新点不在于“从 CT 合成全景片”这一大方向，而在于将生成过程拆分为可解释的几何生成和受约束的模态转换。

首先，本方案没有将 CT 到 PX 直接建模为端到端图像翻译，而是分解为 3D 经验射线几何生成、固定投影合成 SimPX、2D SimPX-PX 模态转换三个阶段。这样能够保留从 CT 到模拟投影的可解释成像链路，也降低了模态转换模型改变解剖结构的风险。

其次，3D 条件流匹配模型按扫描相位生成 CT 空间中的隐式射线几何，而不是依赖固定牙弓曲线或手工设定的全景轨迹。目前已有工作多采用显式轨迹、牙弓曲线、动态旋转中心或已知几何支架，尚未看到与该方案完全同构的患者自适应经验轨迹生成方法。

第三，人工对照真实 PX 标注经验扫描轨迹，再在经验轨迹附近加入受约束扰动，并以扰动强度 $\lambda$ 作为条件建模轨迹邻域分布。这一设计不仅是数据增强，也是在构建“可接受全景轨迹空间”的弱监督样本，使流匹配模型能够学习经验轨迹与偏离轨迹之间的连续变化规律。

第四，2D 流匹配模型只负责 SimPX 到 PX 的灰度分布、模糊、噪声、背景投影和设备后处理特征转换，而不承担主要结构重排。这种分工对于医学影像生成的结构可信度较为重要。

#### 可行性评价

技术上，该方案可行，但关键难点集中在数据构建和标注闭环。

首先应建立非学习版基线：人工交互式标注全景扫描轨迹，使用固定透视叠加算法合成 SimPX，并验证 SimPX 与真实 PX 在主要结构上的一致性。只有当该步骤能够稳定得到结构近似对齐的 SimPX 时，3D 条件流匹配模型才有可靠的弱监督目标。

3D 条件流匹配模型的复杂度需要严格控制。较合理的实现方式是以归一化扫描相位 $t$ 作为条件，每次生成局部相位对应的隐式射线场，而不是一次性生成完整 2000-3000 列全景图对应的全部轨迹体。推理阶段可采用滑动相位窗口以增强相邻相位之间的连续性。

扰动强度 $\lambda$ 作为条件在技术上合理，但应定义为“相对经验轨迹的偏离程度”，而不是“真实程度”。扰动也不能任意添加，应限制为平滑位移、局部角度扰动、焦层宽度扰动、源探测器距离扰动等物理可接受变化，并保持相位单调、轨迹连续、扇束方向稳定和源探测器几何一致。

SimPX-PX 的 2D 流匹配阶段具有较高可行性，但必须通过结构保持约束限制模型自由度。该阶段的目标应是校正真实 PX 的观测域特征，而不是重新生成牙齿、骨皮质或病灶结构。

#### 临床可用性评价

该方案的短期临床定位应是“基于既有 CT 生成临床熟悉的全景样式辅助图像”，而不是直接替代真实 PX 的诊断金标准。

其优势场景较为明确：患者已经完成颌面部螺旋 CT 检查，而临床医生仍希望获得类似全景片的牙弓展开视图，用于观察牙列整体关系、颌骨轮廓、金属修复体、骨缺损、占位范围及与邻近结构的关系。此时，基于 CT 自动生成 PX 样式图像，有望减少额外检查流程和辐射暴露，并提高外科医生对 CT 信息的快速利用效率。

主要临床风险来自两个方面。第一，经验扫描轨迹并不等同于设备真实机械轨迹，生成图像应被视为 CT 派生的全景样式图像。第二，2D 模态转换模型可能制造、弱化或抹除细小结构，因此不能仅以视觉逼真度作为评价标准。

临床验证应以结构一致性和读片任务为核心，而不是仅依赖 SSIM、FID 或专家真假辨别。建议重点评估牙齿编号和位置一致性、阻生牙显示、骨皮质连续性、骨缺损和占位边界、金属伪影区域、下颌管与根尖区域可见性，以及医生基于生成图像和真实 PX 的诊断一致性。

因此，生成图像应明确标注为“由 CT 合成的全景样式图像”，优先用于结构概览、术前沟通和辅助阅片。关键诊断仍应回到原始 CT 及必要的真实影像检查。

#### 总体判断

本方案具有明确创新性，尤其是 CT 空间经验扫描轨迹生成与 SimPX-PX 模态转换的分层设计。其工程实现具有可行性，但高度依赖高质量弱标注、稳定的固定投影基线和严格的结构保持约束。临床上，该方案有清晰应用动机和潜在价值，但需要以结构一致性和读片任务为核心进行验证，避免仅以图像逼真度证明有效性。
