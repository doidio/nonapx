# nonapx

> 口腔颌面外科的低冗余影像生成方案

## Introduction

#### 螺旋 CT 检查的必要性

在大型综合医院及口腔颌面外科中心，螺旋 CT 是颌骨占位性病变患者的常规影像检查。相较于锥形束 CT（CBCT），螺旋 CT 具有更好的软组织分辨率，并可结合增强扫描评估肿瘤浸润范围、血管关系及周围软组织受累情况。CBCT 主要面向高密度骨性结构显示，虽然在牙槽骨、牙根、阻生牙和颌骨骨质评估中具有优势，但软组织分辨率有限，难以准确判断占位性病变边界。因此，在颌骨肿瘤、囊肿及复杂颌面外科病例中，螺旋 CT 往往比 CBCT 更具综合诊断价值。

#### 额外 PX 检查的问题

螺旋 CT 通常不具备牙科全景片（panoramic X-ray, PX）的专用采集功能。患者即使已经完成高分辨率颌面部 CT 检查，仍可能需要额外接受一次 PX 检查，以获得临床医生熟悉的牙弓展开视图、牙列整体关系及颌骨二维全景观察信息。与部分口腔专科设备可在 CBCT/PX 一体机中完成联合采集不同，综合医院场景下的 CT 与 PX 往往来自不同设备、不同时间和不同体位。这不仅增加检查流程和辐射暴露，也导致 CT 与 PX 之间难以建立严格的空间对应关系。

#### 从 CT 生成 PX 的两阶段路径

为了充分利用螺旋 CT 已有的三维结构信息，可以将从 CT 生成 PX 风格全景片的问题拆解为两个阶段。第一阶段是**物理模拟合成**，即在 CT 空间中沿牙弓曲线建立近似全景片采样模型，模拟窄缝扫描、动态旋转轨迹、焦层加权及局部投影关系，生成结构可信的模拟全景片（SimPX）。第二阶段是**图像模态转换**，即将 CT 派生的 SimPX 翻译为更接近真实 PX 设备成像风格的图像。

#### 结构准确性与模态差异

在这一框架中，SimPX 的结构准确性主要由 CT 体数据及显式采样模型保证。由于螺旋 CT 具有较高的空间定位精度，不同合理采样参数生成的 SimPX 均可视为某种可解释的模拟拍摄结果。相比之下，更具挑战的是 SimPX 到真实 PX 的模态转换。真实 PX 与 CT 通常并非同步采集，受头颈姿态、颞下颌关节位置、上下颌开闭状态、牙齿咬合差异及设备成像几何影响，同一患者的 CT 与 PX 之间很难建立像素级配准。因此，即使拥有同一患者的 CT 与 PX，也不应简单视为严格配对数据，而更适合建模为无配对或弱配对的跨模态图像转换问题。

#### CycleGAN 的结构保真局限

无配对图像模态翻译的经典方法包括 CycleGAN 及其变体。此类方法通过双向映射和循环一致性约束，在不依赖成对样本的条件下实现源域到目标域的转换。然而，循环一致性并不等价于解剖结构保真。模型可能在 SimPX 到 PX 的转换过程中改变局部牙齿、牙根或骨性边界，再通过反向网络恢复原始结构，从而满足循环损失，却不能保证中间生成图像在临床结构层面可靠。对于口腔颌面影像而言，这种隐式结构漂移可能掩盖关键解剖差异，限制其作为临床影像替代或辅助工具的可信度。

#### 薛定谔桥的结构保真优势

薛定谔桥（Schrödinger bridge）为无配对跨模态转换提供了另一种建模思路。不同于直接学习确定性图像翻译映射，薛定谔桥将源域分布到目标域分布的转换表述为受熵正则约束的随机最优传输问题，在 SimPX 分布与真实 PX 分布之间寻找一条概率意义下的最优演化路径，尤其适合 CT 生成 PX 的任务要求。SimPX 提供由 CT 保证的结构可信起点，真实 PX 提供目标观测域的成像风格分布，二者之间不要求逐例像素级配对。通过引入结构保持约束，桥接过程可主要学习真实 PX 的模糊、噪声、焦层外投影、设备后处理及灰度风格，同时减少对 SimPX 中牙齿和颌骨结构的不必要变形。

#### 结构保真的分布桥接

因此，CT 生成 PX 可以被定义为结构可信 SimPX 到真实 PX 观测域的结构保持型无配对分布桥接问题。物理模拟模块负责从 CT 中生成具有解剖准确性的 SimPX，生成式模态转换模块负责学习真实 PX 的成像分布。该框架不依赖严格 CT–PX 配准，而是利用 CT 的空间真实性、SimPX 的可解释采样关系以及真实 PX 的域分布特征，探索一种面向口腔颌面外科的低冗余影像生成方案。
